# HB Singapore Banking App

A mobile-first Singapore banking simulation built with React + Vite + TailwindCSS.

## Features
- Home Dashboard with balance & transactions
- Transfer money (HB, PayNow, FAST)
- QR Pay & QR code generation
- Card management (freeze/unfreeze)
- Investment portfolio tracker
- Transaction history
- Firebase Auth (mock mode available)

## Live Demo
**URL:** https://hafeez-learn.github.io/uob-sim-final/

> Demo mode: set `VITE_MOCK_USER=true` in .env for full app without Firebase

## Tech Stack
React 18 + Vite + TailwindCSS + Firebase + Recharts + Lucide Icons
